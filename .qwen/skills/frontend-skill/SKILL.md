# Frontend Skill â€“ Pages, Components & Styling

## Instructions

### 1. Page & Layout Structure
- Use semantic HTML elements
- Mobile-first responsive layouts
- Consistent spacing and grid systems
- Accessibility-first (ARIA, keyboard navigation)

### 2. Component Development
- Reusable and composable components
- Single-responsibility principle
- Clear props and predictable state
- Separate logic from presentation

### 3. Styling
- Modern CSS (Flexbox, Grid)
- Design tokens for colors, spacing, typography
- Responsive breakpoints
- Optional dark/light theme support
- Avoid unnecessary inline styles

### 4. Interactivity
- Clean UI state management
- Subtle transitions and hover effects
- Visual loading and error states
- Fast user feedback

## Best Practices
- Mobile-first approach
- Reuse components
- Consistent UI patterns
- Readable and maintainable styles
- Test across screen sizes
